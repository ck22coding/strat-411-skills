# STRAT 411 Case Analysis Skills

AI-powered case analysis tools built for [Claude Code](https://claude.ai/claude-code), designed around the consulting methodology taught in STRAT 411 (Strategic Analysis) at BYU Marriott School of Business by Professor David J. Bryce.

These skills guide students through a structured 6-phase consulting engagement, from problem framing through deck construction, with built-in research, verification, and presentation workflows.

## The Pipeline

```
411-case          The core skill. 6-phase guided case analysis.
    |
    |-- 411-research    Structured research with SIFT methodology
    |       |
    |       v
    |-- 411-refine      Source verification and confidence scoring
    |       |
    |       v
    |-- 411-reresearch  Targeted replacement of failed sources
    |
    v
411-deck          Produces .pptx deck from case output
```

## Skills

### 411-case (core)

Interactive case analysis agent that walks through 6 phases:

| Phase | Name | What Happens |
|-------|------|-------------|
| 1 | Problem Framing | SCQA framing: Situation, Complication, Question. Industry and firm research. |
| 2 | Solution Development | Generate 3-5 candidates. The selected solution IS the Answer. |
| 3 | Assumption Mapping | QDT stress-test, Pyramid Principle ordering, leadlines, logic tree. |
| 4 | Research Planning | Match assumptions to analysis types, build mock slides, scope data needs. |
| 4.5 | TA Feedback | Checkpoint for instructor/TA review before data collection. |
| 5 | Data Collection | Batched research, verification loops, evidence stitching. |
| 6 | Deck Construction | Apply SCQA + Pyramid to slides. Produce handoff for 411-deck. |

Includes 12 reference files covering strategic theory (rugged landscapes, Schumpeterian competition, dynamic capabilities, platform strategy, competitive positioning) and analytical techniques.

### 411-research

Structured research using SIFT methodology (Stop, Investigate, Find, Trace) with a 4-tier source hierarchy:

- **Gold**: Academic journals, official filings, government data
- **Silver**: Major publications, industry reports
- **Bronze**: Trade press, reputable blogs
- **Noise**: Filtered out

Produces L0/L1/L2 tiered output (executive summary / detailed findings / raw source ledger). Works in standalone mode or receives structured assumption maps from 411-case Phase 4.

### 411-refine

Post-research verification:
- Re-fetches every cited URL and confirms data points appear on page
- Cross-references claims against independent sources
- Assigns Verified/Estimate reliability labels
- Recalibrates confidence scores
- Produces a .docx bibliography

### 411-reresearch

Targeted source replacement when 411-refine finds failures:
- Parses the Verification Dashboard for dead URLs, downgraded claims, single-source findings
- Runs differentiated search strategies per failure type
- Produces replacement sources compatible with 411-refine for re-verification
- Maximum 3 iterations before declaring gaps as UNFILLABLE

### 411-deck

Produces a .pptx presentation deck from a structured handoff spec file (`*-deck-spec.md`) generated by 411-case Phase 6. Fully automated slide production with:
- Layout pattern matching
- Chart construction
- Formatting standards enforcement
- 411-specific QA checks

## Installation

These are [Claude Code skills](https://docs.anthropic.com/en/docs/claude-code). To use them:

1. Install [Claude Code](https://claude.ai/claude-code)
2. Clone this repo
3. Copy each skill folder into `~/.claude/skills/`:

```bash
git clone https://github.com/ck22coding/strat-411-skills.git
cp -r strat-411-skills/411-* ~/.claude/skills/
```

4. Open Claude Code and invoke with `/411-case`

## Usage

Start a case analysis:
```
/411-case [path to case document or description]
```

The skill will guide you through all 6 phases interactively. At each step, it asks what YOU think first, then adds its analysis. It will invoke the research, refine, and deck skills automatically at the appropriate phases.

## Methodology

The analytical framework draws from:
- **Rugged Landscapes** (Levinthal) — NK Model, fitness landscapes, local peak traps
- **Evolutionary Theory** (Nelson & Winter) — Routines, organizational rigidity, bounded rationality
- **Schumpeterian Competition** — Creative destruction, market regimes
- **Dynamic Capabilities** — Meta-routines for radical change
- **Blue Ocean Strategy** (Kim & Mauborgne) — ERRC grid, value innovation
- **Platform Strategy** (Bryce, Dyer & Van Alstyne) — Growth matrix, portfolio coherence
- **Pyramid Principle** (Minto) — Argument structure, leadlines, deductive/inductive ordering
- **SCQA** (Minto) — Situation, Complication, Question, Answer framing
